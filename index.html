О проекте

Бот-проводник на Python, который помогает ориентироваться в новом городе. Определяет местоположение по IP, показывает достопримечательности, кафе и события.

Преимущества

Автоопределение

Никаких ручных настроек - город определяется автоматически

Актуальные данные

Только проверенные места и свежие события

Удобство

Весь функционал доступен прямо в Telegram

Технологии

  • Python
  • Telegram API
  • Geolocation
  • SQL Database
  • REST API

Развитие

  • Интеграция с картографическими сервисами
  • Голосовой помощник
  • Расширение базы городов
  • Система рекомендаций
